Promises

What?

A Promise represents a value which may be available now, or in the future, or never.

Why?

Syntax


      new Promise(
        (resolve,reject)=>{
          //If successful
          resolve(results);

        //If it failed
        reject(reason);
    });

Example

const getExchangeRates=(fail)=>{
      console.log('fetching...');
      return new Promise((resolve,reject)=>{
        setTimeout(()=>{
          if(fail){
            reject("Request Failed 404 Network Error");
          }else{
            resolve({USD:1.0,HKD:0.13});
          }
        },2000)
      });
    }
getExchangeRates(true).then((res)=>{
        console.log(res);
      }).catch((err)=>{console.log('Error: ',err);})
